Network Forensics Platform

An advanced cybersecurity and digital forensics platform designed for automated network traffic analysis, threat detection, IOC extraction, and forensic investigation. The system supports PCAP/log analysis, threat intelligence integration, MITRE ATT&CK mapping, risk scoring, session reconstruction, and real-time monitoring using Python and Flask.

Features
  PCAP, CSV, TXT, and LOG file analysis
  Network traffic forensic investigation
  IOC (Indicators of Compromise) extraction
  Threat intelligence integration
  VirusTotal & Hybrid Analysis support
  Attack detection and correlation
  Risk scoring engine
  MITRE ATT&CK mapping
  Session reconstruction
  Timeline analysis
  Protocol analysis
  TLS fingerprinting
  Tor traffic detection
  Threat hunting dashboard
  Real-time monitoring
  PDF and HTML report generation
  Alerting system
