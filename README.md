Network Forensics Platform

An advanced cybersecurity and digital forensics platform designed for automated network traffic analysis, threat detection, IOC extraction, and forensic investigation. The system supports PCAP/log analysis, threat intelligence integration, MITRE ATT&CK mapping, risk scoring, session reconstruction, and real-time monitoring using Python and Flask.

Features: 
PCAP analysis, CSV/TXT/LOG investigation, IOC extraction, threat intelligence integration, VirusTotal support, Hybrid Analysis integration, attack detection, risk scoring, MITRE ATT&CK mapping, session reconstruction, timeline analysis, protocol analysis, TLS fingerprinting, Tor detection, threat hunting, real-time monitoring, forensic reporting, alerting system.
  
Technologies Used: 
Python, Flask, Flask-SocketIO, Scapy, Pandas, Plotly, ReportLab, HTML, CSS, Bootstrap, Jinja2.
  
Data Processing: 
Pandas, Scapy
  
Frontend: 
HTML, CSS, Bootstrap, Jinja2 Templates
  
Reporting: 
ReportLab, Plotly
  
Threat Intelligence: 
VirusTotal API, Hybrid Analysis API
  
Supported File Formats: 
.pcap, .csv, .txt, .log
  
Project Modules: 
Attack Detection Engine, IOC Extraction Module, Threat Correlation Engine, Risk Scoring System, MITRE ATT&CK Mapping, Session Reconstruction, Timeline Reconstruction, Threat Hunting, Alerting System, Real-Time Monitoring Dashboard
  
Use Cases:
Network forensics, cybersecurity research, incident response, malware investigation, threat hunting, digital investigation, suspicious traffic analysis, academic security projects.
  
Future Improvements:
Machine learning anomaly detection, YARA integration, SIEM integration, Elasticsearch support, Docker deployment, real-time packet sniffing, advanced malware analysis.
